Universal Data Tool Python
================

The Universal Data Tool will allow you to collaborate and edit data within a
Jupyter Notebook. Here's how to get started:

* Install via :code:`pip install universaldatatool`
* Read the `Getting Started Guide <./getting_started.html>`_

.. Tip::

  This documentation and code related to the Universal Data Tool python module is available via `python-universaldatatool on Github <https://github.com/UniversalDataTool/python-universaldatatool>`_
  . You can also access the `Universal Data Tool source code on Github <https://github.com/UniversalDataTool/universal-data-tool>`_.
